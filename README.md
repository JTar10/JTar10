### FINTRAC Rule Monitor
**Python • OpenAI API • Pandas • BeautifulSoup • Git**

Developed a Python automation tool for an anti-money laundering (AML) compliance firm that monitors the validation rules of more than 3,000 FINTRAC fields for regulatory changes. The tool processes validation data and uses the OpenAI API to automatically generate JSON test cases covering valid and invalid inputs, with email notifications when changes are detected.
